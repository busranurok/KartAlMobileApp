# KartAlMobileApp

İnsanların istedikleri kategoride reklam izleyerek kupon kazandığı ve kazandığı bu kuponları kullanarak alışveriş yapabildikleri IOS tabanlı bir mobil uygulama.

# Kullanılan Teknoloji ve Kütüphaneler

- Swift
- CocoaPods
- AVFoundation
